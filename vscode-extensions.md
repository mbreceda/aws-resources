# VS Code Extensions Reference

This file contains a list of all VS Code extensions installed in your environment. Use this list to quickly set up your preferred extensions in a new project or workspace.

## How to Install All Extensions

To install all extensions listed in `vscode-extensions.txt`, run the following command in your terminal:

```bash
cat vscode-extensions.txt | xargs -L 1 code --install-extension
```
